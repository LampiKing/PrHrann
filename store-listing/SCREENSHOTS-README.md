## Priporočeni zasloni za screenshot-e

1. **Iskanje** - Prikaz iskalnika z rezultati (npr. "mleko")
2. **Primerjava cen** - Izdelek z cenami v vseh 3 trgovinah
3. **Akcije** - Seznam trenutnih akcij/popustov
4. **Nakupovalni seznam** - Seznam z izračunanim prihrankom
5. **Začetna stran** - Glavna stran aplikacije
6. **Profil** - Uporabniški profil (opcijsko)

## Feature Graphic (Play Store)
- Velikost: 1024 x 500 px
- Format: PNG ali JPEG
- Vsebina: Logo + slogan "Primerjaj cene. Prihrani."

5. Prvi screenshot mora pritegniti pozornost - naj bo najboljša funkcija
